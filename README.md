# Machine Learning Algorithms — Complete Interactive Guide

An interactive, responsive, and beautifully styled visual dashboard designed to serve as a comprehensive reference guide for core Machine Learning algorithms. Built entirely with clean semantic HTML, custom CSS variables, and native vanilla JavaScript, this single-page application covers everything from foundational theory to practical code implementation.

🎨 **Live Demo:** `mlguide-ten.vercel.app` *(Replace with your actual Vercel link once deployed)*

---

## 🚀 Features

- **Interactive Tabbed Interface:** Seamlessly switch between different paradigms of machine learning (Regression, Classification, Clustering, and Visualizations).
- **Core ML Paradigms Covered:**
  - **Regression:** Linear Regression, Ridge, Lasso, ElasticNet, Decision Trees, Random Forests, Gradient Boosting, XGBoost, and Support Vector Regression (SVR).
  - **Classification:** Logistic Regression, Naive Bayes, KNN, SVMs, Decision Trees, Random Forests, and Boosting variants.
  - **Clustering:** K-Means, Hierarchical (Agglomerative), DBSCAN, and Gaussian Mixture Models (GMM).
- **Deep-Dive Engineering Frameworks:** Distinct conceptual pathways catering to different expertise levels:
  - **Beginner Track:** Focuses on foundational intuition, basic math formulas, key assumptions, and primary hyperparameter definitions.
  - **Advanced Track:** Focuses on complex optimization techniques, mathematical loss formulations, convergence conditions, architectural tradeoffs, and mitigation strategies for underfitting/overfitting.
- **Production-Ready Code Snippets:** Expandable syntax-highlighted code blocks containing implementation workflows utilizing `scikit-learn`, `xgboost`, `numpy`, and `pandas`.
- **Comprehensive Visual Layouts:** Built-in dashboard views showcasing typical assessment configurations (e.g., residual plots, ROC curves, confusion matrices, and decision boundaries) mapped to clear evaluation metrics.

---

## 🛠️ Built With

- **Frontend Architecture:** Semantic HTML5, CSS Custom Variables (Design Tokens), and Vanilla JavaScript (ES6+).
- **Typography:** Google Fonts integration featuring *Syne* for striking headers, *Literata* for highly readable technical prose, and *JetBrains Mono* for exact mathematical and code blocks.
- **Tech Stack Focus:** Documented implementation standard centered around Python ecosystem powerhouses: `scikit-learn`, `xgboost`, `numpy`, `pandas`, and `matplotlib`.
